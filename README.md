## 📖 Resumen Rápido: Los 5 Elementos de un Buen Prompt

```
1. CONTEXTO/ROL → Quién soy en este escenario
   ↓
2. TAREA/OBJETIVO → Qué exactamente quieres
   ↓
3. DETALLES/LIMITACIONES → Restricciones y requisitos
   ↓
4. FORMATO/ESTRUCTURA → Cómo presentar la respuesta
   ↓
5. PREFERENCIAS → Tono, estilo, ejemplos
```

## 💡 Ejemplo Rápido

**Mal Prompt:**
```
"Escribe algo sobre marketing"
```

**Buen Prompt:**
```
Actúa como especialista en marketing digital con 10 años de experiencia.

TAREA: Crea una estrategia de contenido para Instagram

DETALLES:
- Máximo 5 posts por semana
- Formato vertical 9:16
- Dirigido a mujeres 25-40 años
- Objetivo: aumentar engagement 40% en 3 meses

FORMATO: Entrega en 4 secciones:
1. Objetivos
2. Tipos de contenido
3. Calendario
4. Métricas de éxito

PREFERENCIAS: Tono casual pero profesional, incluye emojis, ejemplos de posts reales
```

## 💬 Consejos Finales

1. **Sé específico**: Los detalles mejoran exponencialmente la calidad
2. **Proporciona contexto**: Yo sé más cuando tú me dices más
3. **Define el formato**: Especifica cómo quieres que te presente la respuesta
4. **Revisa antes de enviar**: Un prompt bien estructurado = respuesta perfecta
5. **Itera**: Si no te gusta la respuesta, mejora el prompt y vuelve a intentar
